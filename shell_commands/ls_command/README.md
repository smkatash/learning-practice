# ls command with flag -la & wildcard '*'

List information about the FILEs (the current directory by default).

## Running ls ##

```bash
# Access
$ cd ls_command

# Compile the program
$ make

# Run flags
$ ./ls -a
$ ./ls -l
$ ./ls -la

# Run wildcard '*' in double quotes!
$ ./ls "*"
$ ./ls "*file"
$ ./ls "file*"
$ ./ls "fi*e"

```
